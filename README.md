# Media_Editor_Script
自用脚本。

## bat文件格式
ANSI；GB

## 脚本功能
目前就是给自己用的，所以功能比较简单。
### 应用在视频主体
1、从某个视频文件上CUT出一段的clip脚本；

2、是合并多个视频的merge脚本；

3、转换目录下所有常见封装格式到mp4的X2mp4脚本。

### 应用在图片主体
1、为指定路径下所有图片加边框的border脚本；

2、转换路径下所有图片的高度和宽度为指定大小的resize脚本；

3、转换图片为指定时间长度的视频的pic2vid系列脚本。

## 脚本使用
运行之后有提示词，按照提示词来操作就行，没有设置太多的防呆设计（因为自己设计水平太低所以加入防呆设计可能造成不必要的崩溃）。
