# Media_Editor_Script
自用脚本。

## bat文件格式
ANSI（为了显示中文）

## 脚本功能
目前就是给自己用的，所以功能比较简单。
应用在视频主体上的有两个文件，一个是从某个视频文件上CUT出一段的clip脚本；另一个是合并多个视频的merge脚本。

## 脚本使用
运行之后有提示词，按照提示词来操作就行，没有设置太多的防呆设计（因为自己设计水平太低所以加入防呆设计可能造成不必要的崩溃）。
